# MoreHands Competitor Creative Report

Static HTML report for team review.

## Files

- `index.html` - the report
- `styles.css` - page styling
- `assets/` - screenshots from competitor landing pages and Meta Ad Library

## Preview Locally

Open `index.html` in a browser.

## Free Hosting Outside Netlify

### Can this live on Google Drive or Google Docs?

Sort of, but not as a true website.

- Google Drive is good for sharing the folder or a zipped copy with your team, but it is not a reliable static web host for an HTML page with CSS and image assets.
- Google Docs can publish a document to the web, but it will not preserve this designed HTML report. It works better for a plain memo-style version.
- Google Sites is the best Google-native option. You can create a free Google Site, add the key sections manually, and upload/embed the screenshots. This is the better Google option if your team already lives in Workspace.

For this exact HTML report, GitHub Pages or Cloudflare Pages will preserve the design and screenshots with less friction.

### Option 1: GitHub Pages

1. Create a new GitHub repository named `morehands-competitor-report`.
2. Upload this entire folder's contents.
3. Open repository settings.
4. Go to Pages.
5. Set source to "Deploy from a branch".
6. Choose `main` and `/root`.
7. Share the generated GitHub Pages URL.

### Option 2: Cloudflare Pages

1. Create a free Cloudflare account.
2. Go to Workers & Pages.
3. Choose Pages, then upload assets.
4. Upload this entire folder.
5. Cloudflare will generate a shareable URL.

### Option 3: Vercel

1. Create a free Vercel account.
2. Create a new project.
3. Upload or connect this folder via GitHub.
4. Leave build settings blank because this is plain static HTML.

GitHub Pages is the simplest option for a durable team link.
